Authors and history.

This GNOME game was originally implemented by Miguel de Icaza and Federico Mena at early days of the GNOME project. It was inspired by KDE's samegnome game which in turn was inspired by a Macintosh game of a similar name.

A number of tiles were originated from the KDE project. The planets tile was rendered by Federico in his copious spare time to show the world his masterful control of povray command language.
